# Temporizador Demo Day Achs

Temporizador web sincronizado en tiempo real para eventos Demo Day, con diseño premium basado en el branding de Achs.

## Características
- **Sincronización Real-time**: Usa Gun.js para sincronización descentralizada (sin necesidad de servidor propio).
- **Branding Achs**: Colores institucionales y estética premium.
- **Modos**: Admin (controles totales) y Espectador (solo visualización).
- **Wake Lock API**: Evita que la pantalla se apague durante la demo.

## Uso
1. Abre `index.html`.
2. Selecciona tu rol.
3. Si eres Admin, controla el tiempo. Los cambios se verán en todos los dispositivos conectados.

## Despliegue en GitHub Pages
1. Ve a **Settings > Pages** en este repositorio.
2. En **Build and deployment > Source**, selecciona `Deploy from a branch`.
3. Selecciona la rama `main` y la carpeta `/root`.
4. Haz clic en **Save**.

La URL será: `https://clabza16.github.io/temporizador/`
